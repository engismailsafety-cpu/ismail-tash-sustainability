# 🌍 Global Sustainability Benchmarking Platform

## AI-Powered ESG Analysis for ExxonMobil, Saudi Aramco & BP

### 📌 Overview

This interactive platform provides a professional benchmarking solution to analyze and compare the 2025 sustainability reports of three major energy companies:

- **ExxonMobil**
- **Saudi Aramco**
- **BP**

The platform extracts key ESG metrics, visualizes performance through interactive charts, and delivers predictive insights with a clear winner identification.

### 🚀 Features

- **Secure Login System** (`admin` / `1234`)
- **Multi-File Upload** (3 PDF reports)
- **Ranked Company Cards** with Overall Scores
- **Winner Badge** for Top Performer
- **Progress Bars** for Visual Score Representation
- **Radar Chart** – Compare companies across 6 ESG dimensions
- **Bar Chart** – Side-by-side comparison of key metrics
- **Technical Comparison Table** with best values highlighted
- **Predictive Insights** with Future Outlook (2026-2030)
- **Export to PDF** (Coming Soon)

### 🛠️ Tech Stack

| Technology | Purpose |
|------------|---------|
| **Streamlit** | Web Framework |
| **Pandas** | Data Processing |
| **Plotly** | Interactive Charts |
| **PyPDF** | PDF Text Extraction |
| **ReportLab** | PDF Export |

### 📂 Data Sources

Metrics derived from **2025 Sustainability Reports** of ExxonMobil, Saudi Aramco, and BP.

### 🖥️ How to Use

1. **Login** with `admin` / `1234`
2. **Upload** the 3 PDF reports
3. Click **"Run Benchmark Analysis"**
4. View the **results dashboard**
